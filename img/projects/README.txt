Folder screenshot project
==========================

Taruh screenshot tiap project di sini dengan NAMA FILE PERSIS berikut
(supaya otomatis tampil tanpa perlu mengubah kode):

  web-edukasi.png        -> Project 01 (Web & Interface)
  voucher-hotspot.png    -> Project 02 (Network System)
  server-monitoring.png  -> Project 03 (Server & Monitoring)
  coming-soon.png        -> Project 04 (slot baru, opsional)

Tips:
- Rasio gambar paling pas 16:10 (mis. 1280 x 800 px). Gambar akan
  dipotong otomatis (object-fit: cover), jadi taruh bagian penting di tengah.
- Format .png atau .jpg sama-sama bisa. Kalau pakai .jpg, ganti juga
  ekstensi di index.html (mis. img/projects/web-edukasi.jpg).
- Selama file belum ada, kotak thumbnail menampilkan tulisan
  "Screenshot segera" secara otomatis.

Mengubah link GitHub:
- Buka index.html, cari komentar "Ganti href=..." di atas tiap project,
  lalu ganti https://github.com/mrafiqamin24 dengan URL repository project itu.
- Ada dua href per project (thumbnail dan tombol bulat) -> isi keduanya sama.
